# Ecommerce Website

A responsive front-end ecommerce landing page built with HTML and CSS.

## Features

- Hero section with CTA button
- Featured categories and products
- Latest products grid
- Promotional offer section
- Customer testimonial cards
- Brand/logo showcase
- Footer with app download links and useful links

## Tech Stack

- HTML5
- CSS3
- Google Fonts (Poppins)
- Font Awesome (CDN)

## Project Structure

```text
Ecommerce/
  index.html
  style.css
  images/
  .gitignore
  README.md
```

## Run Locally

1. Clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
